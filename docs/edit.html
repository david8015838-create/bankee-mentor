    🔑 GitHub Personal Access Token

    Token 只儲存在你的瀏覽器 localStorage，不會傳到任何伺服器。
    申請方式：github.com/settings/tokens
    權限只需要：Contents: Read and write（限定本 repo）

    ⚠️ 任何持有此 token 的人都能修改 repo，請勿外流。
